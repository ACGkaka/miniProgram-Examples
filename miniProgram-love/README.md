我们可以在酷狗、酷我等页面上，按 `F12`，搜索 `mp3`，就可以看到相应歌曲的资源地址，复制粘贴到 `` 即可。
> * 酷狗网址：[https://www.kugou.com/](https://www.kugou.com/)
